#Calendar

A beautiful Date Picker with Range and suggestion options

##Dependancies
- jQuery >= 1.7.1
- moment >= 2.8.3
- boostrap >= 3.2.0 working on removing this dependency

##Usage
```js
    $("#date0").calendar()
```

##Options

you can use the date Object anywhere moment is used

###currDate

Default: moment()

```
{
    currDate: moment()
}
```
